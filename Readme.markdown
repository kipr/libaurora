libaurora
=========

libaurora is a graphical user interface for Daylite nodes.

Requirements
============

* CMake 2.8.0 or higher
* libpng
* zlib

Building (OS X and Linux)
=========================

    cd libaurora
    mkdir build
    cd build
    cmake ..
    make
    make install
